Latest Work

HEALTH CLINIC AND PRODUCT FINDERS

This current project includes two different single-page applications to guide customers of an international health brand to the right clinics and products for their needs. They are built using, collectively, AngularJS, Handlebars, JSON, jQuery, Bootstrap 3, HTML and SCSS, integrating the Google Maps API. I am the lead developer on this project. In addition to front-end development alongside our UX team, my roles have included configuring the staging servers using Jenkins, overseeing all architectural decisions, updating the project manager on the team’s progress, and leading weekly dev demos for the client.

SHIFT

Shift is a web application that matches traveling nurses with potential employers. It is built with the LAMP stack using Laravel and Backbone.js, jQuery and SCSS on the front end. My roles in this project have included nearly all front-end enhancements and tasks over the past year, database migrations, CRUD capabilities for the admin panel, CSV exporting functionality, authentication and security features, and hooking up automated email notifications for users.

PORTABLE X-RAY IMAGING TOOL

This imaging tool is distributed to bomb squads for storing and manipulating images associated with suspicious activity. Users can log in to CRUD “incidents”, upload associated images, and use a Photoshop-like interface to adjust each image for a better look. I worked with one other developer on the front-end of this app, mostly in vanilla JavaScript, and my main role was building the histogram window and level sliders that adjust the image’s tones in the editing interface. I also built and styled most of the CRUD and login interfaces using HTML and CSS.

CONSTRUCTION E-COMMERCE SITE

This e-commerce site allows manufacturers to search for, bundle and purchase parts for construction equipment, with helpful guides to make sure they're buying the right combination of items. I wrote modular HTML served up in PHP files, SCSS and JavaScript to build the user interface of this application, including the cart, global navigation, checkout page and users' ‘bundle’ list pages.
